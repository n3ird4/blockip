# blockip

Simple script in order to give you **information about IPs** and the possibility
to **blackhole** them.

prerequisites : 

* curl
* iproute2
* set foldmethod=marker (in my ~/.vimrc)


How to:

* git clone https://github.com/n3ird4/blockip.git
* chmod +x blockip.sh
* sudo cp blockip.sh /usr/bin/blockip
* blockip X.X.X.X
